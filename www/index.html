サッカー部

部員３０名。目標は久々の高専全国大会出場そして初戦突破。近くの高専や高校と試合しています。竹原サッカーリーグに加盟しています。

Page 3

野球部

自主・研鑽をモットーに、自分たちで考えて試合をする「野球が好きな選手達」の集まりです。3年生の選手権大会で高校野球を卒業後は、「もっと野球が好きな選手達」が集まり、5年生の高専大会まで野球を楽しんでいます。

陸上部

本校に入学して初めて競技を始めた者からある程度実績や経験を積んでいる者まで、それぞれの目標に向けて日々練習に取り組んでいます。

卓球部

卓球部は高体連・日学連・高専関係・一般の大会等かなり多くの大会に参加しています。和気あいあいとしたクラブの雰囲気の中、"試合に勝つ"を目標に頑張っています。

Page 3

バレー部

練習は週４日ですが、限られた練習時間で１つでも多くの勝利を目標に頑張っています。未経験者もたくさんいますが、みんなで教え合って上達しています。男女ともに部員の仲が良いのがチームの特徴です。部員はみんな元気で明るく、とても活力に満ちた部活動です。

Page 3

バスケ部

男子バスケ部は40名程度で、出場する試合によって１～３年生チームと、４～５年生中心チームに分かれますが、普段は学年に関係なく楽しく練習しています。女子バスケ部です。女子バスケ部は10名ほどで活動しています。少人数ですが、都合をやりくりして練習しています。

Page 3

漕艇部

私たち漕艇部は全国大会優勝を目標とし、部員一丸となって頑張っています。また、瀬戸内海という恵まれた環境で、季節を肌に感じながら仲間と共に成長できる海のスポーツです。
Page 2
Page 3

柔道部

高等学校体育連盟並びに高等専門学校体育連盟主催の試合(インターハイ関連、高専大会)に加え、全日本柔道連盟・呉地区柔道連盟に加盟し毎月実施される昇段試合・月次試合にも参加し、柔道の技と精神の鍛錬に励んでいます。
Page 2
Page 3

剣道部

剣道部は、週６日で稽古を行っています。高体連や高専の各種大会、他校への遠征、一般の大会にも参加し、技術と精神の向上に努めています。

男子寮

男子棟は１棟・３棟・４棟・５棟と合計４つの棟があり、１人部屋と２人部屋があります。

男子寮の中には捕食談話室・図書室・洗濯室などがあります。

女子寮

女子寮は棟が１つですが１棟に少し女子部屋があります。

女子寮の中には捕食談話室・図書室・洗濯室などがあります。

寮事務

寮事務は主に郵便物を引き取ったり外泊届等の書類を提出したり、他にも救急ボックスやチャリの空気入れなどが置いてあります。

若潮会館

若潮会館は主に長期休暇の際に部活をしている学生が泊まれる合宿所です。捕食談話室や浴室等があります。

長期期間以外にも茶道部や書道部が活動できる場でもあります。
